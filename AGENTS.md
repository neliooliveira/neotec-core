# AGENTS and Skills Guide

## Overview

This document outlines the specialized agents and skills available for NEOTEC Core development.

## Development Agents

### Agent: Backend-Django
**Purpose**: Django development, model creation, API endpoints
**Skills**:
- Django ORM expertise
- DRF serializers and viewsets
- Database migrations
- API design and testing

### Agent: Frontend-React
**Purpose**: Next.js/React development, component creation, UI implementation
**Skills**:
- React hooks and state management
- Next.js app structure
- TypeScript type definitions
- Component composition
- TailwindCSS styling

### Agent: DevOps-Infrastructure
**Purpose**: Docker, deployment, infrastructure as code
**Skills**:
- Docker Compose configuration
- Container orchestration
- Synology NAS deployment
- Caddy reverse proxy configuration
- Database management

## Project Principles for Agents

1. **Security First** - Always validate input, check permissions, use HTTPS
2. **Modular Design** - Keep components/functions small and focused
3. **Testing** - Write tests alongside implementation
4. **Documentation** - Document complex logic and APIs
5. **Performance** - Cache where appropriate, avoid N+1 queries
6. **Dark Mode** - Design for dark theme first
7. **i18n** - Support Portuguese and English

## When to Use Which Agent

- **Backend Work**: Backend-Django agent
- **Frontend Work**: Frontend-React agent
- **Infrastructure/Deployment**: DevOps-Infrastructure agent
- **Mixed**: Use multiple agents with clear handoffs
