# Platform Vision

## Vision Statement

NEOTEC Core centraliza a gestão completa de clientes, projetos, engenharia, produtos, produção, costing e faturação da NEOTEC. Self-hosted em Synology NAS com Docker.

### Core Principles

1. **Security First** - RBAC, auditoria completa, dados isolados
2. **User Experience** - Dark mode first, rápido e intuitivo
3. **Modular Architecture** - Apps independentes, componentes reutilizáveis
4. **Data Integrity** - Histórico completo, versionamento, auditoria
5. **Scalability** - PostgreSQL, caching, async tasks

### Target Users

- Admin (controlo total)
- Manager (gestão de projetos)
- Engineer (trabalho técnico)
- Finance (faturação e custos)
- Client (visão limitada)
- Support (documentação e histórico)

### Key Features

- Gestão centralizada de clientes e projetos
- Requisitos vivos com versioning e costos
- Engenharia integrada (hardware, firmware, software, PCB)
- Produção com encomendas e rastreamento
- Costing detalhado e faturação
- Documentação técnica com metadados
- Auditoria completa de todas as operações
