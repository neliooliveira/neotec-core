# UI/UX Design Principles

## Visual Design

### Color Palette (Dark Mode First)
- **Background**: #0F172A (very dark blue)
- **Surface**: #1E293B (dark blue-gray)
- **Primary**: #3B82F6 (blue)
- **Success**: #10B981 (green)
- **Warning**: #F59E0B (amber)
- **Danger**: #EF4444 (red)
- **Text**: #F1F5F9 (light gray)

### Typography
- Font Family: Inter (sans-serif)
- Headings: Bold, larger sizes
- Body: Regular, readable sizes
- Monospace: Fira Code (for code/IDs)

### Spacing & Layout
- 8px grid system
- 16px gaps between sections
- Max content width: 1200px
- Responsive: Mobile, Tablet, Desktop

## Component Design

- **Buttons**: Primary, secondary, danger variants
- **Forms**: Clean labels, inline validation
- **Tables**: Sortable, filterable, paginated
- **Cards**: Consistent shadow and border radius
- **Modals**: Focused interaction
- **Notifications**: Toast messages, non-intrusive

## Interaction Design

- **Smooth Transitions**: 200-300ms animation
- **Feedback**: Visual feedback on interactions
- **Loading States**: Skeleton screens or spinners
- **Error Messages**: Clear, actionable
- **Success Messages**: Brief, confirmatory

## Localization

- **Portuguese (pt-PT)**: Primary language
- **English (en-US)**: Secondary language
- **Language Selector**: Header or settings
- **All UI strings**: Translatable from day 1

## Accessibility

- **WCAG 2.1 AA**: Minimum compliance
- **Keyboard Navigation**: Full support
- **Screen Readers**: Proper ARIA labels
- **Color Contrast**: 4.5:1 minimum
